# magic-lantern-CR2-sRAW-DNG-Converter
A batch repair tool that converts incompatible Magic Lantern sRAW files (.CR2) into Adobe-ready DNGs.

https://magic-lantern-cr2-sraw-dng-converter.streamlit.app/

A specialized utility for Canon cameras running Magic Lantern. This tool rescues 'Silent Picture' CR2 files that are natively incompatible with Lightroom and Bridge. It automatically detects truncated data, repairs broken files using a reference image, and converts them into standard DNGs for seamless editing.

Magic Lantern CR2-to-DNG Rescuer

Unlock sRAW: Enables usable sRAW workflows on legacy Canon models that lack native support.

Fix Compatibility: Converts non-standard CR2s into DNGs fully compatible with Adobe Lightroom, Photoshop, and Bridge.

Smart Repair: Automatically detects truncated or corrupt files and patches them using data from healthy reference files.

Batch Processing: Drag-and-drop hundreds of files for rapid conversion.
